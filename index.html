Selamat Datang di Web Static Perpustakaan

Website ini merupakan hasil kolaborasi dan dedikasi kami para anggota Library Club. Dan dipersembahkan untuk memudahkan akses informasi dan layanan perpustakaan bagi para pecinta buku di SMAN 8 Tangsel Untuk meningkatkan minat baca dan budaya literasi di sekolah.

Ayo kunjungi website ini dan temukan buku favoritmu!

Jelajahi Koleksi

Koleksi Lengkap

Kami memiliki koleksi buku yang luas, dari buku novel hingga buku ilmu pengetahuan, untuk memenuhi kebutuhan semua pembaca.

Komunitas Pembaca

Bergabunglah dengan komunitas pembaca kami dan berbagi minat serta rekomendasi buku.
